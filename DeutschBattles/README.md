# Deutsch Battles

TELC odakli Almanca calisma uygulamasi. Bu proje React + Vite + localStorage + Gemini ile calisir.

## Kurulum

1. `.env.example` dosyasini `.env` olarak kopyala.
2. Sadece Gemini API key bilgisini doldur.
3. `npm install`
4. `npm run dev`

## Notlar

- Tum uygulama verileri tarayicidaki `localStorage` icinde tutulur.
- Varsayilan model `gemini-3.1-flash-lite-preview` olarak ayarlidir.

## Build

`npm run build`
