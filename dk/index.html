GitHub Pages · Gemini · Ana ekrana eklenebilir (PWA)

Yeni kelime üretmek için önce Ayarlar’dan Gemini API anahtarını kaydedin. Daha önce gösterilen kelimeler tekrar istenmez.

Henüz kart yok — aşağıdan veya Ayarlar sonrası yenileyin.

Şu anki kart

Ana ekrana eklemek (iPhone): Safari’de bu sayfayı açın → paylaş düğmesi → Ana Ekrana Ekle. Kısayol tam ekran açılır. API anahtarı cihazda (tarayıcı depolamasında) kalır; paylaşımlı cihazlarda dikkatli olun.

Ekranınıza gelen kelimeler CEFR seviyesine göre gruplanır; her grupta Almanca alfaya göre sıralıdır.

Google Gemini

Model: gemini-3.1-flash-lite-preview. Anahtar yalnızca bu tarayıcıda saklanır (localStorage).

Google AI Studio — API anahtarı
Anahtar kısıtlamasında HTTP yönlendiricisi olarak sayfanızın adresini ekleyin (ör. https://*.github.io/*/dk/*).
